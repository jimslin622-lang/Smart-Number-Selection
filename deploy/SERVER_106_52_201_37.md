# 服务器 106.52.201.37 部署专用指南

## 🎯 服务器信息

## 🚀 开始部署

### 第一步：上传项目到服务器

在你的 Windows 电脑上，打开 PowerShell 运行：

```powershell
# 进入项目目录
cd C:\Users\84653\Desktop\lottery-app_v1

# 上传压缩包到服务器
scp lottery-app.zip ubuntu@106.52.201.37:/home/ubuntu/
```

**注意**：如果是第一次连接，会提示确认服务器指纹，输入 `yes` 即可。

---

### 第二步：SSH 登录服务器

在同一个 PowerShell 窗口继续：

```powershell
ssh ubuntu@106.52.201.37
```

输入密码后，你就会进入服务器的命令行界面。

---

### 第三步：服务器端部署

在服务器上运行以下命令：

```bash
# 1. 检查文件是否上传成功
ls -lh /home/ubuntu/lottery-app.zip

# 2. 备份旧版本（如果存在）
if [ -d "/home/ubuntu/lottery-app" ]; then
    mv /home/ubuntu/lottery-app /home/ubuntu/lottery-app_backup_$(date +%Y%m%d_%H%M%S)
fi

# 3. 解压新项目
cd /home/ubuntu
unzip -q lottery-app.zip -d lottery-app

# 4. 给脚本执行权限
cd /home/ubuntu/lottery-app/deploy
chmod +x deploy-server.sh
chmod +x rollback-server.sh

# 5. 执行部署
./deploy-server.sh
```

或者直接运行我准备好的部署脚本：

```bash
cd /home/ubuntu/lottery-app/deploy
./deploy-server.sh
```

---

### 第四步：验证部署

```bash
# 进入项目目录
cd /home/ubuntu/lottery-app

# 检查 Docker 容器状态
docker-compose ps

# 查看 API 服务日志
docker-compose logs -f api

# 健康检查测试
curl http://localhost:3000/health

# 检查 Nginx 配置（如需要）
sudo nginx -t
sudo systemctl reload nginx
```
